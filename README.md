# 🔍 Code Analyzer Final

**Code Analyzer Final** — это консольный инструмент статического анализа кода, предназначенный для проверки гибридных файлов, содержащих конструкции C++ и Java-подобные импорты. Анализатор выявляет критические ошибки компиляции и стилистические нарушения, генерируя подробный текстовый отчет.

## 📖 Описание

Утилита `code_analyzer_final` сканирует указанные директории и файлы исходного кода. В отличие от стандартных компиляторов, анализатор специально настроен на поиск специфических проблем:
1. **Синтаксические ошибки**: Пропущенные точки с запятой после `import` или `return`.
2. **Стиль кода**: Имена классов с маленькой буквы, использование глобальных переменных, магические числа.
3. **Структурные ошибки**: Точка с запятой после закрывающей скобки класса `};` (стиль Java в C++ коде).

На выходе формируется отчет `analysis_report.txt` с четкой разбивкой по файлам и итоговой таблицей.

## ✨ Возможности

- ✅ Обнаружение пропущенных `;` в критических местах.
- ✅ Валидация нейминга классов (CamelCase vs lowerCamelCase).
- ✅ Подсветка объявлений глобальных переменных.
- ✅ Парсинг и разделение синтаксиса `import` (Java) и `#include` (C++).
- ✅ Вывод итоговой таблицы со статусом **SUCCESS** или **FAILED**.

## 🛠 Стек технологий

Анализатор написан на языке программирования **C++** и не требует внешних зависимостей для запуска.

| Компонент          | Технология / Стандарт |
| :----------------- | :-------------------- |
| **Ядро парсера**   | C++17                 |
| **Анализ лексем**  | Регулярные выражения (std::regex) |
| **Обход файлов**   | std::filesystem       |
| **Формат отчета**  | Plain Text (UTF-8)    |

## 📄 Структура отчета

Файл `analysis_report.txt` состоит из секций:

1. **Заголовок**: Дата и время запуска.
2. **Секция файла**:
   - Список критических ошибок (`❌ КРИТИЧЕСКИЕ ОШИБКИ`).
   - Список стилистических предупреждений (`⚠️ СТИЛИСТИЧЕСКИЕ ПРЕДУПРЕЖДЕНИЯ`).
   - Блок `=== Code Analysis Report ===` с извлеченными сущностями (Imports, Classes, Methods).
3. **Итоговая таблица**: Сводка по всем обработанным файлам.

## 🚀 Примеры использования

### Запуск анализа
```bash
# Анализ конкретного файла
./code_analyzer_final --path ./test/bad_code_1.cpp

# Анализ всей папки с исходниками
./code_analyzer_final --path ./test/

# Сохранение отчета в кастомный файл (если поддерживается)
./code_analyzer_final --path ./src/ --output ./reports/today.txt
